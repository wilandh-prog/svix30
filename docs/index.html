Methodology. SVIX30 replicates the CBOE VIX calculation on OMXS30 options listed on Nasdaq Stockholm: forwards are estimated per expiration via put-call parity, model-free variance is computed from out-of-the-money bid/ask mid-quotes, and the two expirations bracketing 30 days are interpolated in total variance to a constant 30-day horizon. The single-name indices apply the identical calculation to each equity option class in the same feed; a name is included on days when both the near- and next-term expiration pass the quote-quality gates. Quotes are 15-minute-delayed pre-trade data from Nasdaq Nordic trade reports. This is an unofficial research index, not investment advice.
